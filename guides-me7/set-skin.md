# Как установить скин

Для лицензии:

```
/skin set player никнейм
```

Без лицензии:

* заходим в настройки управления (по умолчанию кнопка Num5), вводим в поиск fabrictailor, назначаем удобную клавишу.
* заходим в gui fabrictailor'a, открываем вкладку&#x20;
